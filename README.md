# Todo_App
A simple and interactive TODO List web app built using HTML, Tailwind CSS, and JavaScript. It allows users to add, edit, delete, and track tasks efficiently.


📌Features:Add new tasks, Mark tasks as completed, Edit existing tasks, Delete tasks, Real-time task counters:, Pending tasks, Completed tasks, Total tasks.


🚀Tech Stack:HTML, Tailwind CSS (CDN), JavaScript.


📍How It Works:


Tasks are stored in a JavaScript array
Each task has:
1. text (task content)
2. completed (true/false)
3. UI updates dynamically using DOM manipulation
4. Event delegation is used for edit, delete, and checkbox actions.
